# Simple Calculator

A simple web-based calculator built using **HTML, CSS, and JavaScript**.
This project performs basic arithmetic operations and includes additional features like keyboard shortcuts, calculation history, and dark mode.

---

## Features

* Addition, subtraction, multiplication, and division
* Percentage calculation
* Power operation (xʸ)
* Delete last input
* Keyboard shortcut support
* Calculation history
* Dark / Light mode toggle
* Responsive and simple user interface

---

## Technologies Used

* **HTML** – Structure of the calculator
* **CSS** – Styling and layout
* **JavaScript** – Calculator logic and functionality

---

## Keyboard Shortcuts

| Key       | Function          |
| --------- | ----------------- |
| Enter     | Calculate result  |
| Backspace | Delete last value |
| S         | Square root       |
| ^         | Power operation   |
| D         | Toggle dark mode  |
| H         | Open history      |
| C         | Clear display     |

---

## How to Run the Project

1. Download or clone the repository.
2. Open the project folder.
3. Run the `index.html` file in any web browser.
